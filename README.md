# DOOM-SVG
It's DOOM, ported to a SVG
## How do I run this?
- Running precompiled
    - Go to https://simpansoftware.cc/doom-svg/doom.svg on a pc, and that's DOOM!
    - Alternatively open doom.svg from this repo
- Compiling and then running
    - Requirements: Python
    - Clone the repo, go to build and run the following command
    ```python3 build.py```
    - After that, open up doom.svg in the build folder in the browser and you should be running DOOM
## Controls?
- Arrow keys or WASD to move
- Space to interact with doors etc
- CTRL or Left Mouse button to shoot (note: mouse movement doesnt work)
